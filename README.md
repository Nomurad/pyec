# PyEC  

PyEC is a framework for evolutionary computing in Python3.  

## Algorithms
* NSGA-II
* MOEA/D
* MOEA/D-DE
* Constraint-MOEA/D
* CMOEA/D-DMA
* CMOEA/D-DEDMA
* CMOEA/D-HXDMA

## Install  
    python -m pip install -e .

## requirement  

* python3.6
* numpy
